# Awesome-chocholate-sales-dashboard(SQL and Power BI)
##  Project Objective
To analyze sales performance using SQL and Power BI by transforming raw data into actionable insights. To identify key trends, growth patterns, and regional contributions that support data-driven decision-making.

## Questions(KPI)
1. What is the total revenue generated during the analysis period?
2. How many boxes were sold overall?
3. How many shipments were made?
4. What percentage of products were low box shipment?
5. Which regions contributed the most to total sales?
6. Which product categories generated the highest sales?
7. Which salesperson contributed the most to revenue?
8. How did sales trend over time?
